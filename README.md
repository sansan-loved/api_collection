# API 收藏夹

一个现代化的API管理工具，帮助开发者轻松管理和组织API接口。

## 功能特点

### 核心功能
- 📁 **API管理**：添加、编辑、删除API接口
- 🏷️ **标签系统**：按标签分类和筛选API
- 🔄 **多视图切换**：卡片视图和表格视图
- 🔍 **搜索功能**：实时搜索API
- 💾 **导入导出**：数据备份和恢复
- 📱 **响应式设计**：适配不同屏幕尺寸

### 特色功能
- 🔐 **API Key保护**：默认隐藏API Key，保护敏感信息
- 🎨 **自定义标签**：支持创建自定义标签和颜色
- 📊 **标签计数**：显示每个标签下的API数量
- 🔄 **API移动**：可以将API移动到其他标签下
- 🌙 **深色主题**：现代化的深色界面设计

## 技术栈

- **前端**：HTML5, CSS3, JavaScript (ES6+)
- **存储**：localStorage (本地存储)
- **字体**：Noto Sans SC, JetBrains Mono
- **图标**：Font Awesome

## 快速开始

### 方法一：直接打开

1. 下载项目文件到本地
2. 直接打开 `index.html` 文件即可使用

### 方法二：本地服务器

1. 下载项目文件到本地
2. 在项目目录下启动本地服务器：

   ```bash
   # 使用Python
   python -m http.server 8000
   
   # 或使用Node.js
   npx serve
   ```

3. 打开浏览器访问 `http://localhost:8000`

## 使用指南

### 添加API
1. 点击顶部导航栏的「新增API」按钮
2. 填写API信息：标题、请求方式、URL、API Key（可选）、标签、描述、文档链接
3. 点击「保存」按钮

### 管理标签
1. 点击左侧边栏的「新建标签」按钮
2. 填写标签名称，选择颜色，添加描述（可选）
3. 点击「保存」按钮

### 移动API
1. 点击API卡片或表格中的移动按钮（箭头图标）
2. 选择目标标签
3. 选择操作方式：
   - 添加到新标签（保留原标签）
   - 移动到新标签（移除原标签）
4. 点击「确认移动」按钮

### 搜索API
在顶部导航栏的搜索框中输入关键词，系统会实时筛选API。

### 导入导出数据
- **导出**：点击顶部导航栏的「导出」按钮，数据会下载为JSON文件
- **导入**：点击顶部导航栏的「导入」按钮，选择之前导出的JSON文件

## 数据存储

所有数据都存储在浏览器的localStorage中，不会上传到任何服务器。请定期使用导出功能备份数据，以防浏览器数据丢失。

## 开发说明

### 代码结构
- **HTML**：使用语义化标签，结构清晰
- **CSS**：使用CSS变量，便于主题定制
- **JavaScript**：使用ES6+语法，模块化设计

### 核心类
- `ApiCollection`：主应用类，管理API和标签

### 主要方法
- `renderApiList()`：渲染API列表
- `renderTagList()`：渲染标签列表
- `saveData()`：保存API数据
- `saveTags()`：保存标签数据
- `moveApi()`：移动API到其他标签
